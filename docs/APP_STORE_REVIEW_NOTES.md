# App Store Review Notes

Paste into App Store Connect → App Review Information → Notes.

```
Donna is a personal AI second-brain for iPhone.

SIGN IN
• Use Sign in with Apple (preferred) or Sign in with Google.
• After sign-in, accept the one-time AI data consent screen to enter the app.

HOW TO TRY THE CORE FEATURE
• Open the Chat tab and tap the microphone to speak. Donna streams audio to our
  production backend, transcribes, replies, and can speak back.
• Notes, Memory, Actions, and Today tabs show saved content and suggested work.
• Profile lets you change persona/model, export data, or delete the account.

OPTIONAL HARDWARE
• “Pair Donna device” (Bluetooth) is optional. The app is fully usable with the
  phone microphone alone. Reviewers do not need a Donna Device.

ACCOUNT DELETION (Guideline 5.1.1)
• Profile → Delete account → confirm.
• This permanently removes the Supabase auth user and associated server data.
```
